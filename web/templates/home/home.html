Hello, world!

This is a (very) simple AngularJS single-page application that aims to help beginners with their first API-consumer app.

Simple Form

Here we have a (very) simple form, to show the power of a Controller's $scope. If you already have filled any of the fields, you may click on the "Show user summary" button to see a styled summary appearing on the second half of the page.

Simple List/Filter

This is pretty much the simplest you can get to list stuff using AngularJS. By clicking on "Populate List", the app will make a GET request to jsonplaceholder/posts and fetch the full 100 post list. Also, here we have a simple filter per column, that uses AngularJS' "| filter:{}" functionality.

User "CRUD"

Here we have a user list, with the filters working the same way we have on Simple List, although now you'll be able to "Edit" an User. For that, we're using URL parameters and we fetch the user by id on-demand.

To-do List

An example of how to implement a to-do list using AngularJS. By changing the select, the "limit" value will be changed on the $scope, also changing the "limitTo" filter value. All of the data is provided by jsonplaceholder.
